## Office Control 

Turn on/off lights/outlets based on light switch, time of day, day of week, motion and contact sensors

## File Structure

* smartapp.js &mdash; the SmartApp implementation
* locales/en.json &mdash; English version of the app configuration page text

## Node Package Dependencies
1. npm i @smartthings/smartapp --save
2. npm i @katesthings/smartcontrols --save
